# Модуль `text` — Отображение текста в DotcLUI

Этот модуль упрощает вывод текста в LÖVE2D с поддержкой позиционирования, шрифтов, цветов и базовой валидации входных данных.

---

# Установка

Убедитесь, что файл модуля расположен по пути:

**DotcLUI/ui/text/text.lua**

---

# Подключение

Чтобы подключть любой модуль просто добавьте это в самое начало

```Lua
local DotcLUI = require("DotcLUI/init.lua")
```

---

# **Основные функции**

## 1. text.der(текст, x, y)
Простой вывод текста — удобен для отладки или быстрого отображения. 

текст (string или любой тип) — значение, которое будет преобразовано в строку и выведено.
- x (number, необязательно) — координата по горизонтали (по умолчанию: 100).
- y (number, необязательно) — координата по вертикали (по умолчанию: 100).

`Пример:`
```Lua
DotcLUI.ui.text.der("HELLO!", 100, 200)
```

## 2. text.new(data)
**Гибкий способ отображения текста** с поддержкой шрифтов, цвета и точного позиционирования. 

| Поле        | Тип         | Описание    |
|-------------|-------------|-------------|
| text или [1]| любой       | **Обязательно**, Текст для отображения, будет преобразован в строку  |
| xy или [2]  | таблица     | Координаты: `{x = 100, y = 200}` или `{100, 200}`.    |
| font        | строка      | Путь к файлу шрифта (только `.ttf`) |
| size        | число       | Размер шрифта (работает только если указан `font`)|
| color       | таблица     | последним значением указывается тип цвета такие как: rgb, rgba, love|

- Важно: если вы указали `font`, но не указали `size`, шрифт будет применёе но в размере 20
- Цвет типа love принимает от 0 - 1 как в стандартном `love.graphics.setColor()`


`Пример:`
```Lua
DotcLUI.ui.text.new({
    "HI"
})

DotcLui.ui.text.new({
    "HI", {100, 200}, 
    color = {1, 1, 1, "love"},
    font = "DotcLUI/style/font/Gasalt Regular.ttf",
    size = 40
})

DotcLui.ui.text.new({
    xy = {100, 200}, 
    text = "HI"
    color = {255, 125, 25, 200 "rgba"},
    font = "DotcLUI/style/font/Gasalt Regular.ttf",
    size = 40
})
```

## Важная информация

1. Если поле `text` отсутствует или равно `nil`, модуль автоматически выведет **Caution: Bad Argument**
2. Все координаты приводятся к числу. При ошибке используется значение по умолчанию (100, 100)
3. Цвет по умолчанию — белый с полной непрозрачностью (`{1, 1, 1, 1}`)
4. Разбор входных данных защищён с помощью `pcall`, поэтому критические ошибки не остановят игру. (Недоработано)
5. Если в функции `text.new({"text"})` оставить только один аргумент то текст выведится на координатах (100, 100)
6. Если вы будете использовать функции класса `text` (`text.new()` и `text.der()`) за пределами `function love.draw()` то ничего отображатся не будет
7. На данный момент есть встроенный шрифт *Gasalt Regular.ttf* он находится по пути `DotcLUI/style/font/Gasalt Regular.ttf`, можете использовать его
